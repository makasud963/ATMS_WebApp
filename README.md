# ATMS_WebApp

Spring Boot Backend Starter Project with:

- User Registration
- Login API
- JWT Authentication
- Spring Security
- Audit Table
- Log File Storage
- Swagger Integration
- Lombok Support

## Run Project

mvn spring-boot:run

## API Endpoints

### Register API
POST:
http://localhost:9097/auth/register

### Login API
POST:
http://localhost:9097/auth/login

## Swagger URL
http://localhost:9097/swagger-ui/index.html

## Log File
logs/atms-webapp.log
